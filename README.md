# 🚀 局域网快传助手 (Local LAN Transfer) V2.0 Pro

一个基于 Python + Flask + WebSocket 打造的高颜值、现代化的局域网文件与文本互传工具。彻底告别数据线和微信文件传输助手，实现设备间的极速、无缝协同。

## ✨ V2.0 工业级架构升级

* **🔐 动态 PIN 码安全网关**：彻底杜绝公共 Wi-Fi 下的隐私裸奔！每次启动自动生成 4 位动态 PIN 码，客机扫码必须凭密码访问；**主机端智能识别，免密直通**，兼顾极客级安全与丝滑体验。
* **🚀 超大文件切片传输 (Chunked Upload)**：引入企业级文件处理架构。前端自动进行 2MB 极速切片，后端流式拼合，完美支持 GB 级别超高清视频/大型压缩包的稳定传输，彻底告别浏览器卡死与内存溢出。
* **📱 PWA 原生应用体验**：完美支持 iOS/Android 的“添加到主屏幕”功能。告别浏览器地址栏，享受拥有独立图标、毛玻璃沉浸式全屏的系统级原生 App 体验。
* **🍎 Apple 级设计美学**：采用现代化毛玻璃 (Glassmorphism) 与便当盒 (Bento Grid) 布局，引入丝滑的阻尼动画与密码锁遮罩，颜值即正义。
* **⚡ 全双工实时同步**：基于 Socket.IO 的事件驱动架构，文本与文件双端零延迟秒级同步，告别手动刷新。
* **🛡️ 阅后即焚生命周期**：支持动态设置文件销毁时效（1小时 / 24小时），到期自动静默清理，释放本地磁盘空间。

## 🛠️ 安装与运行指南

### 前置要求
* 需安装 [Python 3.8+](https://www.python.org/) 环境。
* 确保所有互传设备连接在**同一个 Wi-Fi（局域网）**下。
