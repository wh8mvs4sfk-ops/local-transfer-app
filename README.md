# 🚀 局域网快传助手 (Local LAN Transfer)

一个基于 Python + Flask + WebSocket 打造的高颜值、现代化的局域网文件与文本互传工具。彻底告别数据线和微信文件传输助手，实现设备间的极速、无缝协同。

## ✨ 核心特性

* **🍎 Apple 设计美学**：采用现代化毛玻璃 (Glassmorphism) 与便当盒 (Bento Grid) 布局，UI 极其优雅。
* **⚡ 零延迟实时同步**：基于 WebSocket (Socket.IO) 的全双工通信，文本与文件双端秒级同步，告别手动刷新。
* **📱 扫码即连**：自动获取本机局域网 IP 并生成二维码，手机端扫码即可沉浸式全屏使用。
* **🛡️ 阅后即焚 (TTL 机制)**：创新的文件生命周期管理，支持动态设置文件销毁时效（10秒~7天），到期自动静默删除，保障隐私不留痕。
* **🔌 网络状态感知**：工业级的断网重连检测与优雅的 UI 提示。

## 🛠️ 安装与运行指南

### 前置要求
* 需安装 [Python 3.8+](https://www.python.org/) 环境。
* 确保所有需要互传的设备连接在**同一个 Wi-Fi（局域网）**下。

### 1. 下载项目与安装依赖
将本项目克隆或下载到本地，然后在项目目录下打开终端，执行以下命令安装必要依赖：
